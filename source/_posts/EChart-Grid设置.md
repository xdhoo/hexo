# 设置绘图区域(Grid)的两种方式

## 通过 top / right / bottom / left 

![图1](/img/echart/grid01.png)

```
grid:{
    top:'3%',
    right'3%',
    bottom:'3%',
    left:'3%',
    width:'800px',
    height:'400px'
  },
```

## 通过 x / y / x2 / y2 

![图2](/img/echart/grid02.png)
```
grid:{
    x:'10px',
    y:'10px',
    x2:'10px',
    y2:'10px',
    width:'800px',
    height:'400px'
  },
```
